第一节 随机变量的概率分布（复习）

随机变量

定义

随机试验 — 可在相同条件下重复进行、事先知道可能出现的结果、但试验前不知道哪个结果会发生。

定义： 随机变量 — 用数值描述随机试验结果的变量，常用 \(X, Y, Z\) 表示。

类型 特点 例子
离散型 取值可一一列举 次品个数、掷骰子点数
连续型 取值不能逐个列举，取某区间内任一点 使用寿命、身高

离散型随机变量的概率分布

概率函数：\(P(X = x_i) = p_i\)

期望（集中程度）：

$$\mu = E(X) = \sum x_i \cdot p_i$$

方差（分散程度）：

$$\sigma^2 = D(X) = \sum [x_i - E(X)]^2 \cdot p_i$$

两点分布

随机变量 \(X\) 只取 0 和 1 两个值。

  • 期望：\(E(X) = p\)
  • 方差：\(D(X) = p(1-p)\)

二项分布

定义

在 \(n\) 次独立重复伯努利试验中，成功的次数 \(X\) 服从二项分布，记为 \(X \sim B(n, p)\)。

伯努利试验条件： 1. 一次试验只有两个可能结果（"成功"和"失败"） 2. 成功概率 \(p\) 对每次试验都相同 3. 试验相互独立，可重复 \(n\) 次

$$P(X = x) = C_n^x \cdot p^x \cdot (1-p)^{n-x}$$

$$E(X) = np, \quad D(X) = np(1-p)$$

泊松分布

用于描述在指定时间范围或空间范围内事件出现次数的分布。

$$P(X = x) = \frac{\lambda^x e^{-\lambda}}{x!}$$

$$E(X) = \lambda, \quad D(X) = \lambda$$

连续型随机变量

  • 取值不能逐个列举，用概率密度函数 \(f(x)\) 描述
  • 取任何特定值的概率为 0，通常研究区间概率
  • 密度函数曲线下面积 = 1

分布函数 \(F(x) = P(X \leq x)\)： - \(F(x)\) 是 \(f(x)\) 的积分 - \(f(x)\) 是 \(F(x)\) 的导数 - \(F(x)\) 单调非减

均匀分布

随机变量在 \([a, b]\) 内取任何等长区间的概率相等，记为 \(X \sim U[a, b]\)。

$$E(X) = \frac{a+b}{2}, \quad D(X) = \frac{(b-a)^2}{12}$$

正态分布

定义

连续型随机变量中最重要的分布，记为 \(X \sim N(\mu, \sigma^2)\)。

特点： - 对称钟形曲线，均值 = 中位数 = 众数 - \(\mu\) 决定位置，\(\sigma\) 决定形状 - 曲线下总面积 = 1

标准正态分布：\(Z \sim N(0, 1)\)

$$Z = \frac{X - \mu}{\sigma}$$

例子

零件长度 \(X \sim N(10, 0.2^2)\)。求 \(P(X < 9.4)\) 和 \(P(9.5 < X < 10.5)\)。

\(P(X < 9.4) = P(Z < \frac{9.4-10}{0.2}) = P(Z < -3) \approx 0.00135\)

\(P(9.5 < X < 10.5) = P(-2.5 < Z < 2.5) \approx 0.9876\)

由正态分布导出的三大分布

分布 构造 用途
\(\chi^2\)（卡方） \(n\) 个独立标准正态变量的平方和 方差估计、拟合优度
\(t\) 分布 \(Z / \sqrt{\chi^2 / \nu}\) 小样本均值推断
\(F\) 分布 两个独立 \(\chi^2\) 变量之比 方差分析、回归检验

第二节 概率抽样方法

基本概念

  • 总体（Population）：研究对象全体，分有限总体和无限总体
  • 样本（Sample）：按抽样原则从总体中抽取的部分单位
  • 概率抽样：按随机原则抽取，每个单位被抽中概率已知
  • 随机样本：由概率抽样所抽取的样本

概率抽样的特点

特点 说明
随机原则 每个单位都有一定机会被抽中
概率已知 可以计算出来
避免主观偏差 防止样本代表性不足
可推断总体 基于大数定律

简单随机抽样

从总体 \(N\) 个单位中随机抽 \(n\) 个，每个容量为 \(n\) 的样本有相同机会被抽中。

  • 操作方法：抽签法、随机数表法
  • 抽取方式：重复抽样、不重复抽样
优点 局限性
简单直观 \(N\) 大时不易构造抽样框
估计方便 单位分散，实施困难

抽样框

定义

包括研究对象所有单位的一个名单或框架。确定抽样框后，可计算每个个体的抽样概率。

分层抽样

按某种特征划分为层（层内差异小），每层独立随机抽样。

  • 优点：样本结构与总体结构相近，提高估计精度

整群抽样

将总体单位合并为群，直接抽群，中选群全部调查。

  • 与分层区别：整群抽样扩大"总体单位"，只需群的抽样框

系统抽样

按一定顺序排列，随机起点后等间隔抽取。

  • 随机起点等距抽样、半距起点等距抽样、对称等距抽样

多阶段抽样

大规模调查中多次整群抽样的嵌套。先抽群，再在群内抽单位。

第三节 抽样分布

总体分布、样本分布、抽样分布

这三个概念是统计推断的基石，必须严格区分：

定义

总体分布 — 总体中所有个体的观测值所形成的分布。参数是固定的（\(\mu, \sigma\)），通常未知。

样本分布 — 从总体中抽取的一个具体样本中 \(n\) 个观测值的分布。每次抽样得到的样本不同，分布也不同。

抽样分布 — 某个样本统计量（如 \(\bar{x}\)）在所有可能样本中的概率分布。它是统计推断的理论基础。

总体分布 样本分布 抽样分布
数据来源 全体个体 一个样本的 \(n\) 个值 所有可能样本的统计量
性质 固定但未知 随样本变化 有确定的理论分布
均值 \(\mu\) \(\bar{x}\) \(E(\bar{x}) = \mu\)
标准差 \(\sigma\) \(s\) \(\sigma/\sqrt{n}\)（标准误差）
例子

设某大学 10000 名学生的身高为总体，\(\mu = 168\)cm，\(\sigma = 10\)cm。

总体分布：10000 名学生的身高分布（\(\mu = 168, \sigma = 10\)）。

样本分布：从中随机抽取 50 名学生，这 50 人的身高分布。

$$\bar{x} \text{ 可能是 } 167.3, \quad s \text{ 可能是 } 9.8$$

抽样分布：如果反复抽取无数个容量为 50 的样本，计算每个样本的均值 \(\bar{x}_1, \bar{x}_2, \bar{x}_3, \ldots\)，这些样本均值的分布就是抽样分布。

根据中心极限定理：

$$\bar{x} \sim N\left(168, \frac{100}{50}\right) = N(168, 2)$$

抽样分布的标准差 \(\frac{10}{\sqrt{50}} \approx 1.41\) 称为标准误差，比总体标准差（10）小得多——这就是为什么用样本均值估计总体均值比较精确。

样本均值的抽样分布

从均值为 \(\mu\)、方差为 \(\sigma^2\) 的总体中抽容量为 \(n\) 的样本：

条件 抽样分布
正态总体，\(\sigma\) 已知 \(\bar{x} \sim N(\mu, \frac{\sigma^2}{n})\)
正态总体，\(\sigma\) 未知，小样本 \(\frac{\bar{x} - \mu}{s/\sqrt{n}} \sim t(n-1)\)
非正态总体，大样本 \(n \geq 30\) 近似 \(N(\mu, \frac{\sigma^2}{n})\)（中心极限定理）

📝 小测验：样本均值的抽样分布标准差被称为？

  • 总体标准差
  • 标准误差
  • 样本方差
  • 离散系数

中心极限定理

定理

从均值为 \(\mu\)、方差为 \(\sigma^2\) 的任意总体中抽容量 \(n\) 的样本，当 \(n \geq 30\) 时，样本均值的抽样分布近似正态：

$$\bar{x} \sim N\left(\mu, \frac{\sigma^2}{n}\right)$$

样本均值的标准误差：\(\sigma_{\bar{x}} = \frac{\sigma}{\sqrt{n}}\)

📝 小测验：当样本量 n ≥ 30 时，从任意总体中抽取的样本均值近似服从什么分布？

  • t 分布
  • 正态分布
  • F 分布
  • 卡方分布

样本比例的抽样分布

当 \(np \geq 10\) 且 \(n(1-p) \geq 10\)（大样本）：

$$p \sim N\left(P, \frac{P(1-P)}{n}\right)$$

两个样本均值之差的抽样分布

  • 两大样本 → 近似正态
  • 小样本 + 正态 + 方差相等 → \(t(n_1 + n_2 - 2)\)

两个样本比例之差的抽样分布

  • 两大样本 → 近似正态

本章小结

  1. 随机变量分离散型和连续型；正态分布是最重要的连续分布
  2. 二项分布 \(E(X)=np\)，\(D(X)=np(1-p)\)；泊松分布期望=方差=\(\lambda\)
  3. 标准正态分布 \(Z \sim N(0,1)\) 可通过 \(Z = \frac{X-\mu}{\sigma}\) 转换
  4. 三大抽样分布：\(\chi^2\)（方差）、\(t\)（小样本均值）、\(F\)（方差分析）
  5. 概率抽样：简单随机、分层、整群、系统、多阶段
  6. 中心极限定理：\(n \geq 30\) 时样本均值近似正态
  7. 抽样分布是参数估计和假设检验的理论基础
